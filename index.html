La structure de données tableau

Les structures de données possèdent souvent de nombreuses variantes. Je présente ici l'une des variantes.

Définition et propriétés.

Un tableau est une structure de données permettant de mémoriser plusieurs informations dont les propriétés classiques sont :

    Ces informations sont de même type (int, float, string...) L'accès à l'une des informations se fait instantanément en donnant sa position dans le tableau La modification de l'une des informations se fait instantanément Un tableau est mutable

D'autres propriétés peuvent être ajoutées si le besoin s'en fait sentir :

    Obtenir la taille du tableau instantanément Agrandir ou réduire la taille du tableau instantanément …

Implémentation en Python

Python dispose d'une structure de données de type tableau dynamique appelée list qui peut d'utiliser comme tableau.

Création d'un tableau en Python

Pour créer un tableau en Python, on peut utiliser trois méthodes :

    Méthodesyntaxe<Exemple À l'aide de la répétition *nomDuTableau = [None]*nbValeurstableau = [None] * 1000 Par énumération des valeursnomDuTableau = [valeurs séparées par des virgules]tableau = [3, 1, 4, 1, 5] Par compréhensionnomDuTableau = [formule(k) for k in range(debutInclu, finExclu)]pairs = [2*k for k in range(10)]

Accès à l'une des cases du tableau

Pour accéder à la case numéro k du tableau tab, on utilise la syntaxe valeur = tab[k]. Attention, les indices, positions, rangs, numéros des cases du tableau commencent à 0…

Modification de l'une des cases du tableau

Pour modifier la valeur de la case numéro k du tableau k, on utilise la syntaxe tab[k] = nouvelle valeur
